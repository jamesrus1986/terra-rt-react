# react-native-terra-rt-react

React Native bridge for Terra Realtime mobile SDKs

## Installation

```sh
npm install react-native-terra-rt-react
```

## Usage

```js
import { multiply } from 'react-native-terra-rt-react';

// ...

const result = await multiply(3, 7);
```
